# Student Information System

## Project Overview

The Student Information System is a simple console-based Java application developed to manage student records.
This project was created as part of the Developer's Arena Internship Program to demonstrate fundamental Java programming concepts such as classes, objects, ArrayList, and user interaction through a menu-driven system.

The application allows users to store and manage student details including Student ID, Name, Age, Grade, and Contact Information.

---

## Features

* Add new student records
* View all students
* Search for a student by Student ID
* Input validation for age and grade
* Menu-driven user interface
* Organized object-oriented code structure

---

## Technologies Used

* Java
* ArrayList (Java Collection Framework)
* Scanner (for user input)
* Object-Oriented Programming Concepts

## How to Compile and Run

Open terminal inside the **src folder** and run the following commands:

Compile the program:

javac *.java

Run the program:

java StudentInformationSystem
